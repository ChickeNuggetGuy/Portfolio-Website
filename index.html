Projects

SunRun

Sun Run is an open-world survival game where players must complete dynamic objectives while staying cool and avoiding the scorching summer sun. Featuring a Multi-body-part Sun Exposure System, players are challenged to strategically navigate the environment to protect their character's health. The game includes a Procedural Suburban House Generator, a Modular Quest System, and integrated Health and Stamina Mechanics.

Grid Based Strategy Game

Grid Based Strategy Game is a 3D, grid-based, turn-based tactics game heavily inspired by the original 1994 X-COM, built from the ground up in Godot using GDScript. The game features a Procedural Terrain Generator to create varied battlefields, a Custom 3D A* Pathfinding Implementation tailored for non-flat environments,a grid based inventory System with decoupled data and visuals, and a Modular Action System that supports extendable unit abilities. It also includes a Line-of-Sight System for tactical cover interactions and a Destructible Environment System that reacts to combat decisions. Designed with scalability and readability in mind, the codebase demonstrates efficient handling of game logic and grid calculations without relying on heavy instantiation.

Git Repository: Grid Game Repository

Arena Shooter

Arena Shooter is a fast-paced, class-based prototype developed in Unreal Engine 5.6. The game features Three Distinct Playable Classes with unique attributes like Rate of Fire, Movement Speed, and Special Abilities. It includes Fully Functional AI Opponents, a Complete Game Loop, and a Prototype UI that supports player feedback and class switching. This prototype emphasizes modularity and balance, serving as a solid foundation for a scalable competitive shooter experience.

Git Repository: Arena Shooter Repository
